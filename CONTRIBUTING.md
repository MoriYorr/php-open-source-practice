# Как внести вклад

## Как запустить проект
1. Клонируйте репозиторий
2. Убедитесь, что установлен PHP 8.0+
3. Файлы не требуют установки зависимостей (пока)

## Требования к коду
- Соблюдайте стандарт [PSR-12](https://www.php-fig.org/psr/psr-12/)
- Используйте строгую типизацию
- Документируйте публичные методы с помощью PHPDoc

## Как отправлять pull request
1. Создайте ветку от main
2. Реализуйте функциональность
3. Убедитесь, что код проходит синтаксическую проверку (`php -l`)
4. Откройте PR в main с описанием изменений

## Как писать коммиты
Следуйте [Conventional Commits](https://www.conventionalcommits.org/):
- chore: для служебных задач
- feat: для новой функциональности
- fix: для исправлений
- docs: для документации

Пример: \`feat(calculator): add subtract method\`
